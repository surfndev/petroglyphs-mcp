How it Works

✍️

Write

Draw naturally with Apple Pencil on your iPad. Math, diagrams, notes — anything you'd put on paper.

⚡

Send

One tap sends your handwriting to any LLM. No screenshots, no switching apps.

🔗

Connect

Works with Claude Desktop, Cursor, and any MCP-compatible client on your computer.

Getting Started

  1. Sign up for the beta

    Fill out the form below to get the TestFlight link for the iPad app.

  2. Download the iPad app

    Install from TestFlight and open Petroglyphs on your iPad.

  3. Open your LLM client on computer, add the MCP server

    In Claude Desktop: Settings → Connectors → Add custom connector → Add

    Name: Petroglyphs
    Remote MCP server URL: https://web-production-d8b49.up.railway.app/mcp

  4. Start a new chat and ask for a pairing key

    Type get pairing code from petro Claude will generate a pairing code to link your iPad to this session.

  5. Insert the key in your iPad app

    In Petroglyphs app: Server → Cloud → Paring code → Test Connection → Save

  6. Draw and send

    Write on the canvas and tap to send. Use these commands in your chat:

    get draw from petro · get pairing code from petro

  1. Install the MCP server

    Follow the instructions on GitHub to set up the server locally, or run npx petroglyphs-mcp --port 3333.

  2. Configure your LLM client

    Point Claude Desktop or Cursor to the running MCP server.

  3. Download the iPad app

    Install from TestFlight and enter your computer's local IP address in the app.

  4. Draw and send

    Write on the canvas, tap to send, and see the AI respond on your computer. Both devices need to be on the same Wi-Fi.
